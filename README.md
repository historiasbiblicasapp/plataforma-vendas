# Plataforma de Vendas Digitais Premium

Este é o projeto inicial da plataforma baseada na arquitetura proposta (Next.js + Vendas via WhatsApp).

## Como Rodar Localmente

1. Navegue até a pasta do projeto no seu terminal:
   ```bash
   cd c:\src\Projetos\web\plataforma-vendas
   ```
2. Instale as dependências (caso falte alguma):
   ```bash
   npm install
   ```
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

Acesse [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado do design UI/UX com Neon Premium.
